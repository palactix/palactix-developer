# Introduction

Palactix is a publishing API where you bring your own OAuth apps.

Palactix provide a unified API for publishing content to multiple social platforms while using your own OAuth applications.

Your clients authorize **your application**, not a third-party vendor. OAuth tokens belong to you. Client connections are portable.

**Supported Platforms:** Meta (Facebook), Instagram, LinkedIn, TikTok, X (Twitter), Youtube

---

## The Core Difference

**Traditional tools:**
```
Agency → Hootsuite → Client's Instagram
         ↑
    (Hootsuite owns token)
```

**Palactix:**
```
Agency → Your Meta App → Client's Instagram
         ↑
    (You own token)
```

**Result:** Switch tools anytime without reconnecting accounts.

## Example Request

```bash
POST /v1/publish
Authorization: Bearer your_token

{
  "account_id": "acc_123",
  "content": {
    "text": "Product launch!"
  }
}
```

One publishing API works across multiple platforms.

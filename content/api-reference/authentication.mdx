# Authentication

All API requests must include a Bearer token in the `Authorization` header.

---

## Header Format

```bash
Authorization: Bearer plx_abc123...
```

---

## Example Request

```bash
curl -X GET /v1/accounts \
  -H "Authorization: Bearer plx_abc123..."
```

---

## Notes

- Token must be valid and not expired  
- Missing or invalid tokens will return `401 Unauthorized`  
- Always send token in header, not in query params  

---

## Common Errors

### 401 Unauthorized

Occurs when:

- Token is missing  
- Token is invalid  
- Token is expired

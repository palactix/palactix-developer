# Base URL

All API requests must be made to the following base URL.

---

## Production

```bash
https://api.palactix.com
```

---

## Versioning

Palactix APIs are versioned via the URL.

```bash
https://api.palactix.com/v1
```

---

## Full Endpoint Example

```bash
POST https://api.palactix.com/v1/oauth/token
```

---

## Notes

- Always prefix endpoints with `/v1`  
- Future versions (`/v2`, `/v3`) will be released without breaking existing integrations  
- Do not hardcode internal service URLs
